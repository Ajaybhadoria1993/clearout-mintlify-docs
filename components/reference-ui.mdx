## FAQ Accordion

<Accordion title="What is the Chrome Extension?">
  The Clearout Chrome Extension allows you to verify and validate email
  addresses directly in your browser.
</Accordion>

<Accordion title="How do I install the extension?">
  You can install the extension from the Chrome Web Store by searching for
  "Clearout" or <a href="https://chrome.google.com/webstore">clicking here</a>.
</Accordion>

<Accordion title="Is it free to use?">
  The Chrome Extension is free to install, with in-app purchases available for
  advanced features.
</Accordion>
